Introductory Tutorial¶

Geo-Schelling Model

In this tutorial we will build a Geo-Schelling model, which takes the seminal Schelling Segregation model and replicates with the countries of Europe, were the countries “move” their preference similar to the Schelling Model.

Users can use Google Colab or Jupyter Lite to use this tutorial.

This tutorial uses JupyterLite to allow users to immediately access and adjust code.

This tutorial also uses Jupyter Bridge so users can see more detailed explanations of each code cell if desired by clicking the book icon.
